    IMPORTANT : pour déployer la gem nested form sur les vms en mode production, il faut :
        - inclure la gem dans le gem file,
        - faire un bundle install,
        - retirer la balise <%= javascript_include_tag :defaults, "nested_form" %> dans app/views/layout/application.html.erb,
        - rajouter dans app/assets/javascripts/application.js  --> //= require jquery_nested_form
        - lancer la commande bundle exec rake assets:precompile (commande à exécuter à chaque modifs des assets (js, css, nouvelle image etc..))
        - si problème pour l'installation, se référer à la page suivante --> https://github.com/ryanb/nested_form/
    
    
    IMPORTANT, pour les MIGRATIONS, sur les vm en mode production :
        - pour appliquer les migrations sur une vm en mode production, on execute la commande
         rake db:migrate, suivi de, RAILS_ENV=production, pour spécifier que l'on souhaite appliquer les migrations à la base crm_production
